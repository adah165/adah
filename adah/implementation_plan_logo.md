# خطة عمل: دمج شعار منصة أداة (ADAH) الجديد بشكل احترافي ومتجاوب مع المظهر الداكن

تحتوي هذه الخطة على تفاصيل دمج الشعار الجديد لمنصة أداة (ADAH) في الموقع برمجياً عبر تحويله إلى SVG ذكي يتكيف مع المظهر الداكن (Dark Mode) والمظهر الفاتح (Light Mode)، وتوزيعه في الأماكن المناسبة (Favicon، القائمة الجانبية، وصفحة تسجيل الدخول والإنشاء).

## متطلبات التصميم الفني للشعار (SVG Design Specs)

1. **الأيقونة (الرمز البصري):**
   - يتكون من شكلين متداخلين على هيئة حرف **A** مع سهم صاعد للأعلى واليمين.
   - **اللون الأزرق (شكل الشريط المنحني):** رمز انسيابي يعبر عن المرونة والتحكم (درجة اللون: `#75b5e9` أو `sky-400`).
   - **اللون الأخضر (شكل السهم والقاعدة):** رمز صاعد يعبر عن النمو والحماية (درجة اللون: `#86d7ab` أو `emerald-300`).
   - خلفية شفافة تندمج بسلاسة مع لوحة التحكم الغامقة والفاتحة.

2. **النص برمجياً (Wordmark):**
   - كلمة "Adah" باللغة الإنجليزية أو "أداة" باللغة العربية.
   - استخدام لون تعبئة متجاوب `currentColor` مع تخصيصه برمجياً عبر Tailwind (مثلاً `text-slate-800 dark:text-slate-200`) لضمان عدم حدوث أي تضارب مع الدارك ثيم.

## التعديلات المقترحة (Proposed Changes)

### 1. مكون الشعار الموحد (Shared Logo Component)

#### [NEW] [Logo.tsx](file:///d:/ADS/adah/components/ui/Logo.tsx)
- إنشاء مكون React مرن تحت اسم `Logo` يدعم:
  - `variant`: عرض الأيقونة فقط (`icon`)، أو الشعار الكامل مع النص عمودياً (`full`)، أو الشعار الكامل مع النص أفقياً (`horizontal`).
  - `size`: حجم الأيقونة (العرض والارتفاع بـ pixels).
  - `textClassName`: كلاسات Tailwind لتنسيق النص وتغيير لونه برمجياً (افتراضياً: `text-slate-800 dark:text-slate-200`).
  - `lang`: تحديد لغة النص المعروض (`ar` لعرض "أداة" أو `en` لعرض "Adah").
