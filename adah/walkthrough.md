# دليل المراجعة والتسليم (Walkthrough) - صفحة سياسة الخصوصية لـ Google OAuth

تم إنشاء صفحة سياسة الخصوصية (Privacy Policy) باللغة العربية ونشرها تحت مسار `/privacy` لتلبية متطلبات جوجل الخاصة بـ **OAuth Consent Verification** لصلاحية الوصول الحساسة `https://www.googleapis.com/auth/adwords`.

## ما تم إنجازه (What was accomplished)
1. **إنشاء صفحة سياسة الخصوصية باللغة العربية:**
   - تم إنشاء ملف [page.tsx](file:///d:/ADS/adah/app/privacy/page.tsx) وهو عبارة عن React Server Component متجاوب تماماً.
   - يدعم كلاً من الوضع الداكن (Dark Mode) والوضع الفاتح (Light Mode) بشكل تلقائي.
   - تم إعداد البيانات الوصفية (Metadata/SEO) لتكون صديقة لمحركات البحث ومراجعي جوجل.
   
2. **تغطية متطلبات جوجل وشروط الحماية بشكل صريح:**
   - **البيانات التي نجمعها:** معرفات إعلانات جوجل (Customer ID, Campaign ID)، تفاصيل تكوين الحملات، وعناوين الـ IP الخاصة بالنقرات لكشف الاحتيال.
   - **الغرض من الاستخدام:** قراءة وإدارة الحملات الإعلانية، وحظر النقرات الاحتيالية واستبعاد الـ IPs المشبوهة عبر الـ API لحماية ميزانية الإعلان.
   - **عدم مشاركة البيانات:** تعهد صريح ومميز بعدم بيع أو تأجير أو مشاركة أي بيانات يتم الحصول عليها من Google APIs مع أطراف ثالثة أو معلنين آخرين.
   - **حماية البيانات وسرية المعلومات:** تشفير البيانات المخزنة والتواصل عبر HTTPS/TLS المشفر.
   - **تحكم المستخدم وحذف البيانات:** توضيح كيفية إلغاء التفويض عبر إعدادات جوجل، بالإضافة إلى إمكانية طلب الحذف الكامل لكل البيانات المخزنة بمراسلة البريد الإلكتروني المعتمد `support@adah.sa`.

## كيفية التحقق والتجربة (Verification & Manual testing)

نظرًا لوجود قيود على خادم التشغيل في بيئة التطوير الخاصة بالأداة (Access Denied لـ `agentapi.bat`)، نرجو منك تشغيل الخادم والتحقق يدوياً كما يلي:

1. **تشغيل خادم التطوير:**
   قم بفتح منفذ الأوامر في مجلد المشروع وتشغيل:
   ```bash
   npm run dev
   ```

2. **تصفح مسار الخصوصية:**
   توجه إلى الرابط التالي في المتصفح:
   [http://localhost:3000/privacy](http://localhost:3000/privacy)

3. **التحقق البصري والتجاوب:**
   - تأكد من ظهور الصفحة باللغة العربية وتنسيق خطوط Cairo و Tajawal.
   - جرب تصفح الصفحة على وضع الجوال للتأكد من التجاوب التام.
   - جرب النقر على زر "الرجوع للرئيسية" للتأكد من أنه يوجهك بشكل صحيح.

4. **تطابق البيانات:**
   تطابق البنود تماماً مع دليل متطلبات مراجعة جوجل (Google Ads API Production Checklist) كما هو موثق في [google_ads_production_checklist.md](file:///d:/ADS/adah/google_ads_production_checklist.md).
